Questionário de Desenvolvimento Web

1. Explique a diferença entre Internet e Web e como elas se relacionam.

A Internet é a infraestrutura física de rede, a “espinha dorsal” que conecta computadores no mundo todo. Pense nas ruas e rodovias. Já a Web (ou World Wide Web) é um dos serviços que roda sobre essa infraestrutura, sendo um sistema de documentos interconectados por hiperlinks. É como o tráfego de carros que se move pelas ruas.

2. O que é um website e como ele difere de uma aplicação web?

Um website é basicamente um conjunto de páginas com conteúdo estático ou pouco dinâmico. Um site institucional ou um blog são bons exemplos. Uma aplicação web, por outro lado, é um software que roda no navegador e permite interações complexas com o usuário, como cadastros, processamento de dados e funcionalidades em tempo real. Pense no Google Docs ou no Gmail.

3. Quais são as principais tecnologias client-side e qual a função de cada uma?

No front-end, as principais tecnologias que rodam no navegador do cliente são:

  • HTML: A base de tudo. É a linguagem de **marcação** que define a estrutura do conteúdo da página.
  • CSS: Cuida da parte visual. Ele **estiliza** o HTML, definindo cores, layout, fontes, etc.
  • JavaScript: Dá vida à página. Adiciona **interatividade**, como animações, validação de formulários e manipulação de elementos.

4. Por que o HTML é chamado de linguagem de marcação e não de programação?

O HTML é uma linguagem de **marcação**, pois usa "tags" para descrever e estruturar o conteúdo. Ele não tem lógica de programação, como variáveis ou funções. O papel dele é apenas dizer o que é o que na página (um título, um parágrafo, uma lista, etc.).

5. Descreva a estrutura básica de um documento HTML5 e a função de cada parte.

A estrutura de um documento HTML5 é bem simples:

  • <!DOCTYPE html>: A primeira linha, avisa ao navegador que é um HTML5.
  • <html>: O elemento raiz que engloba tudo.
  • <head>: Aqui ficam os metadados da página, como o título e a ligação com arquivos CSS.
  • <body>: Todo o conteúdo visível para o usuário fica aqui.

6. Qual a função da instrução <!DOCTYPE html> em um documento HTML?

O <!DOCTYPE html> é uma declaração que instrui o navegador a renderizar a página seguindo o padrão HTML5. Sem ela, o navegador poderia entrar no "modo de compatibilidade" e apresentar a página de forma inesperada.

7. Diferencie as funções das tags <head> e <body> em um documento HTML.

O <head> é para informações "invisíveis" ao usuário, como o título da página (<title>), links para folhas de estilo e scripts. Já o <body> contém tudo o que é visível: textos, imagens, botões, etc.

8. Explique a importância do charset UTF-8 na criação de páginas web.

O charset UTF-8 é um padrão de codificação de caracteres universal. Ao usá-lo, garantimos que acentos, cedilhas e caracteres especiais de diferentes idiomas sejam exibidos corretamente no navegador, evitando aqueles símbolos estranhos no lugar do texto.

9. O que são tags HTML e quais os componentes básicos de um elemento HTML?

Tags são os marcadores usados para criar elementos (<p>, <a>). Um elemento é formado pela tag de abertura, a tag de fechamento e o conteúdo. Exemplo: no elemento <p>Olá, mundo!</p>, a tag de abertura é <p> e a de fechamento é </p>.

10. Quais as diferenças entre as tags de cabeçalho (h1 até h6) e qual sua importância para mecanismos de busca?

As tags de cabeçalho <h1> a <h6> definem a hierarquia e a importância do conteúdo. O <h1> é o mais importante e deve ser usado para o título principal da página. Essa hierarquia é crucial para os **mecanismos de busca**, que a usam para entender a estrutura e a relevância do conteúdo.

11. Descreva os principais usos da tag <p> e a diferença entre <p> e <br>.

A tag <p> cria um **parágrafo**, agrupando um bloco de texto. Ela adiciona um espaçamento natural antes e depois. A tag <br>, por outro lado, é apenas uma **quebra de linha** simples, usada para forçar o texto a ir para a próxima linha sem iniciar um novo parágrafo.

12. Explique a diferença entre uma lista ordenada (<ol>) e uma lista não ordenada (<ul>).

Uma lista ordenada (<ol>) é usada quando a ordem dos itens importa, como um passo a passo. Ela exibe os itens com números. Uma lista não ordenada (<ul>) é usada para itens sem uma sequência específica, como uma lista de compras, e exibe os itens com marcadores.

13. O que é o atributo alt da tag <img> e qual sua importância?

O atributo alt fornece um texto alternativo para a imagem. Ele é fundamental para a **acessibilidade**, pois leitores de tela o usam para descrever a imagem para pessoas com deficiência visual. Além disso, o texto é exibido se a imagem não carregar.

14. Qual a função da tag <a> e quais os atributos mais usados nela?

A tag <a> cria hiperlinks. Seu atributo mais importante é o href, que aponta para o destino do link. O atributo target="_blank" é muito usado para abrir o link em uma nova aba.

15. O que é semântica no HTML5 e cite três elementos semânticos introduzidos nesta versão.

A **semântica no HTML5** é o uso de tags que dão significado ao conteúdo. Ao invés de usar apenas <div>, usamos tags como <header>, <nav> e <footer> para descrever a função de cada seção da página. Isso melhora a acessibilidade e o SEO.

16. Explique a diferença entre <section>, <article> e <aside>.

<section> é um agrupador temático, como um capítulo em um livro. <article> é um conteúdo independente e completo, como uma postagem de blog. <aside> é um conteúdo relacionado, mas secundário, como uma barra lateral de links.

17. Qual a função da tag <form> no HTML e como ela se relaciona com aplicações web?

A tag <form> cria um formulário para coletar dados do usuário. Ela é o ponto de partida para enviar informações ao servidor, que as processará e gerará uma resposta. É a base da interação entre o usuário e a aplicação web.

18. Compare os métodos GET e POST em formulários HTML.

O método GET envia os dados pela URL, sendo rápido, mas inseguro para dados sensíveis. Já o método POST envia os dados no corpo da requisição, tornando-o mais seguro e sem limites de tamanho, ideal para senhas e uploads de arquivos.

19. O que é o CSS e quais problemas ele resolve em relação ao uso apenas do HTML?

O CSS é uma linguagem de estilo que define a aparência do HTML. Ele resolve o problema de ter que estilizar cada elemento individualmente com atributos. Com CSS, podemos separar a estrutura do conteúdo da sua apresentação visual, facilitando a manutenção e a consistência do design.

20. Quais são os três tipos básicos de seletores CSS e como eles funcionam?

Os três tipos de seletores mais usados são:

  • Seletor de Tipo: Aplica o estilo a todos os elementos de um tipo (ex: p { ... }).
  • Seletor de Classe: Seleciona elementos com um atributo class específico (ex: .card { ... }).
  • Seletor de ID: Seleciona um único elemento com um id exclusivo (ex: #logo { ... }).
