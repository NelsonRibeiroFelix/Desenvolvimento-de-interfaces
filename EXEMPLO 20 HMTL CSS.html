Exemplo 18 HTML - CSS

Visão geral sobre CSS

CSS é um acrônimo para folhas em estilo cascata. Essa técnica visa personalizar, estilizar um documento HTML. Para isso, usamos regras CSS, que representam uma estrutura que define como os elementos selecionanados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor Indica quais elementos serão afetados pelo estilo definido.
  • Bloco de declaração O bloco composto por propriedades e valores, esse bloco será aplicado aoa elemento HTML.

Seletor de ID

O seletor de ID é aquele que aplica a regra CSS a um componente HTMl que tenha um atributo ID no componente HTML. Este seletor de ID deve ser aplicado a um único elemento HTML, haja vista que o ID deve ser único. Este seletor tem prioridade sobre os demais seletores (de classe e de tipo)

Este seletor sempre começara com um " # ", seguido pelo nome do seletor e suas propriedades e valores.

Neste exemplo usamos seletores de classe e de tipo para estilizar o documento HTML.

Seletores de ID

Os seletores de tipo são aqueles que são representados pelo nome da tag do elemento HTML, seguido de chaves {propriedade: valor;}

Neste exemplo estamos utilizando seletores de tipo para estilizar o documento HTML.
