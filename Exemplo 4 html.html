Exemplo 4 HTML - Estrutura geral de um documentoHTML

Este exemplo mostra a estrutura minima de um documento html. Aqui estão as principais tags utilizadas:

  • <meta name="viewport" content="width=device-width, initial-scale=1.0" >:Essa tag é crucial para tornar a página responsiva, ou seja, adaptável a diferentes tamanhos de tela (dispositivos móveis, tablets, computadores).

  • < width=device-width >: Define a largura da página para corresponder à largurado dispostivo.

  • < initial-scale=1.0 >: Define o nível de zoom inicial da página como 100%.

  • < meta charset="UTF-8 >:UTF-8 é uma codificação universal que suporta a maioria dos caracteres e símbolos utilizados em diferentes idiomas, incluindo português. Isso garante que acentos, cedilhas e outros caracteres especiais sejam exibidos corretamente na página.

  • < titleDocumentTitle >:Essa tag define o título da página, que é exibido na barra de título do navegador ou na aba da página.

  • <!DOCTYPE html >: Essa declaração informa ao navegador que o documento segue o padrão HTML5. É essencial para que o navegador interprete e renderize a página corretamente.

  • <html >:A tag html é o elemento principal do documento html, ela defini o container raiz de toda página

  • <head >:A tag head contém os metadados que serão interpretados pelo navegador. Nela encontraremos o título da pagina e links arquivos para configuração do documento

  • <body >:a tag body é responável pela exibição do conteúdo visível do documento. ou seja , tudo que estiver neste bloco será visível para o usuários
