Exemplo 4 HTML - Estrutura geral do documento HTML

A estrutura geral de um documento HMTML é composta pelas tags estrturantes html, head e body. Essas tags junto como outras são interpretadas pelo navegador que as renderiza para exibir conteúdos aos usuários. A seguir descrevemos os principais elementos do documento HTML

  • <meta name="viewport" content="width=device-width, initial-scale=1.0" >:Essa tag é crucial para tornar a página responsiva, ou seja, adaptável a diferentes tamanhos de tela (dispositivos móveis, tablets, computadores).

  • < width=device-width >Define a largura da página para corresponder à largurado dispostivo.

  • < initial-scale=1.0 >Define o nível de zoom inicial da página como 100%.

  • < meta charset="UTF-8 >UTF-8 é uma codificação universal que suporta a maioria dos caracteres e símbolos utilizados em diferentes idiomas, incluindo português. Isso garante que acentos, cedilhas e outros caracteres especiais sejam exibidos corretamente na página.

  • < titleDocumentTitle >Essa tag define o título da página, que é exibido na barra de título do navegador ou na aba da página.

  • <instrução!DOCTYPE html >A intrução !DOCTYPE HTML é reponsavel por dizer ao navegador qual a versão do html ele deverá usar. Ela é uma instrução e ão uma tag html. Ela deverá vir sempre no topo da página, ja na primeira linha. Quando não informamos esta instruçãoo navegador decice qual versão utilizar.

  • <html >A tag html é a tag raiz do documento html, ela envolve todo conteúdo da página. Indica o inicio e o fim do documento html. Ela pode ainda incluir o atributo LANG que define o idioma que será utilizado.

  • <head >O elemento HEAD contém os metadados sobre a página. Está tag é uma tag de cabeçalho, utilizada pelo navegador para configurações e renderizações do documento HTML. Nela também teremos a tag title que é responsável pelo único elemento visível para o usuário fora os elementos visíveis que são mostradas na tag body.

  • <body >A tag body inclui todo o conteúdo visível do documento HTML, tais como texto, imagens, links, vídeos, botões dentre outros elementos. É neste bloco que estará toda a interação com o usuário.
