Flexbox ou Grid Layout ?

O Flexbox e o CSS Grid Layout posem ser usados em conjunto. você pode aplicar o flexbox a elementos dentro de um layout de grade, aproveitando o melhro dos dois mundos.

A escolha entre Flexbox e Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu layout e escolher técnica que melhor atende as essas necessidades.
