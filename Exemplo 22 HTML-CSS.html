cores cores e estilos de fundo

uso do beckground com sistemas de cores com rgb e rgba

sistemas de cores rgb

os sistema de cores rgb representa um esquema de cores baseado nas cores r(vermelho) g(verde) a(azul), cada parametro define a intencidade da cor e ela varia de 0 a 255, por exemplo rgb(255,99,71)

sistema de cores rgba

o sistema de cores rgba sao uma extençao dos valores dos valores rgb com um canal alfa (a), que especifica a opacidade da cor que pode variar de 0-1 sendo o 0 totalmente invesivel e o 1 otalmente opaco, por exemplo rgba(35,99,71,0.3)
