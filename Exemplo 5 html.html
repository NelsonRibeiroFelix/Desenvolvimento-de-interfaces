Exemplo 5 HTML - tags de seção e Títulos

Os principais elementos HTML incluem tags para estrutura do documento HTML bem como tags específicas para renderização de texto, midias, arquivos de uma forma geral. A seguir descrevemos alguns desses elementos.

Tag P - parágrafo

A tag "P" dentro do sinal de diamante (< >) é utilizado para representar parágrafos em documentos HTML. Esta tag esta sendo usada neste momento para renderizar este conteúdo como um parágrafo deste documento.

TAG BR

A tag "BR" é utilizada para quebras de linha, e podemos utilizar quantas tags forem necessárias. Neste exemplo usamos três tags "BR" para separa esse parágrafo do próximo

Tag "HR"

A tag "HR" é uma tag utilizada para criação de uma linha horizontal que ocupa toda a largura da página, neste exemplo usamos essa linha para separar esse parágrafo do próximo
