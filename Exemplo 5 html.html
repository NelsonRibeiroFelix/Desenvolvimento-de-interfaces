os principais elementos html incluem tags para estrutura do documnetos bram como tags especificas para renderização de texto, midias, arquiso de uma forma geral. A seguir descrevemos alguns desses elementos

Tag p - paragrafo

A tag p dentro do sinal de diamante e utilizado para representar paragrafos em documentos html. essa tg esta sendo usada neste momento para rederizar este conteudo como um paragrafo deste documento

Tag br

A tag br e utilizada para a quebra de linha e podemos usar quantas tags forem necessarias. Nesse Exemplo usamos tres tags br para separar esse paragrafo do proximo

TAG hr

a tag hr e usada para a ação de criação de uma linha horizontal que ocupa toda a largura da página, nese exemplo usamos essa linha para separar este paragrafo do proximo
