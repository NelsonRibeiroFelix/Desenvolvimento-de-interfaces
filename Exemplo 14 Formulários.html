Exemplo 13 HTML - Formulários

Os fomrulários são umas principais formas de se interagir com aplicações web. Esses formlários utilizam diversos componentes que melhorarão a experiência do usuário.

Para trabalhar com formulários devemos usar a tag &alt;form> em conjunto com outras tags que vão representar os componentes de formulários, tais como: labels(rotúlos), inputs (caixa de texto), botões, elemntos de checagem, de seleção entre outros componentes. Neste exemplo, demonstraremos o uso dos métodos Get e Post

Métodos do protocolo HTTP

Get

Quando trabalhamos com a criação de formulários HTML, nós deveremos na maior parte das vezes utilizar o protocolo de rede HTTP, esse protocolo possui vários métodos que são utilizados para facilitar o processo de comunicação entre o cliente e o servidor.

O metodo GET é aquele que que anexa os dados do formulário a uma URL, esses dados do formulário são visiveis na barra de endereço no navegador.

É importante saber que este métodos não é recomendado para operações de inclusão, pois os dados vão aparecer para qualquer pessoa, o que pode ser inseguro.

Post

Já o método POST é aquele que envia os dados para o servidor no corpo da página. Essas informações não são visiveis para o cliente sendo um pouco mais seguras que quando utilizamos o método GET.
